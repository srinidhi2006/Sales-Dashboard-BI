# Dynamic Sales Dashboard using Power BI

## Overview
An interactive Power BI dashboard built on sales data to analyze:

- Product performance
- Quantity ordered
- Revenue trends
- Profit analysis
- Monthly sales trends

## Features

✅ Dynamic slicers

✅ Cross filtering

✅ Custom tooltip pages

✅ KPI Cards

✅ Interactive Pie Charts

✅ Matrix reports
